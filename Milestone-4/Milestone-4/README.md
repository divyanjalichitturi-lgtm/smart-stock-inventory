# Smart Stock Inventory
## Infosys Springboard Virtual Internship – Milestone 4

### 📌 Features
- Inventory Dashboard
- Flask REST APIs
- SQLite Database
- Alerts for Low Stock
- Full-Stack Integration

### 🛠 Technologies Used
- HTML, CSS, JavaScript
- Python (Flask)
- SQLite
- GitHub

### ▶️ How to Run the Project

#### Step 1: Run Backend
```bash
cd backend
pip install -r requirements.txt
python app.py

Step 2: Run Frontend
Open frontend/index.html in your browser or use Live Server.
🌐 API Endpoints
/api/dashboard
/api/products
/api/alerts
